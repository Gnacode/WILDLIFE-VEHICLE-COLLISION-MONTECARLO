04 Sensitivity sweep: Baseline detection rate κ

Baseline per-frame detection rate κ. Lower κ stresses the system with degraded sensor sensitivity (e.g. weather, vegetation occlusion, hardware aging). The cumulative detection probability is 1 − exp(−κ · t_exposure), so even at low κ the system can recover detection if animals dwell long enough in range.

05 Methods summary

All results were generated by a single Python simulator, wvc_simulator.py, using a behaviorally-realistic discrete-time Monte Carlo framework on a 1 km test corridor.

System architecture

Alternating-side Doppler radar nodes at 15 m spacing (50 sensors/km baseline, 67/km at 15 m), three-axis magnetometer sites every 200 m, and dynamic message signs (DMS) every 250 m. Sensors communicate over an LVDS+LoRa hybrid backbone; an awareness signal propagates between neighboring radars to boost detection sensitivity by a factor of 1.8× during 30 s windows around recent detections.

Vehicle dynamics

Treiber–Hennecke–Helbing Intelligent Driver Model (IDM) with desired gap s₀ = 5 m, time headway T = 1.5 s, comfortable acceleration a = 2.5 m/s², comfortable deceleration b = 4.0 m/s², emergency braking 9.0 m/s². Driver reaction delay 1.5 s.

Animal behavior

Six-state Markov model: Foraging → Approaching → Hesitating → {Crossing | Frozen | Fleeing} → Gone. Behavioral transitions calibrated against field studies: p(freeze | vehicle present) = 0.10, p(flee | vehicle) = 0.20, p(cross | clear) = 0.80, p(freeze during cross | vehicle braking) base 0.15. Animals arriving in a slow-vehicle environment cross more often but freeze less.

Three operating modes

  • Control — no sensors, no alerts, no caution speed.
  • Detection — sensors active, drivers receive DMS warnings and reduce to caution speed when in any active awareness zone, no neighbor-sensor boost.
  • Aware — Detection mode plus LoRa-propagated awareness boost (1.8× sensitivity on neighboring radars for 30 s after each detection).

Statistical analysis

All p-values from two-sided Welch's t-tests on per-trial collision counts. Significance markers: *** p < 0.001, ** p < 0.01, * p < 0.05, n.s. otherwise. Error bars and ± uncertainties throughout are one sample standard deviation (Bessel-corrected).

06 Data and reproducibility

All figures and tables are generated directly from the CSVs by this repository's analysis script. Re-running the script on the raw data reproduces every number on this page byte-identical.

Source CSVs

  • wvc_results_backup2.csv — headline three-mode dataset
  • sweep_sweep_spacing.csv — sweep over radar_spacing
  • sweep_size.cs — sweep over size_scale
  • sweep_detection_sweep_detection_rate.csv — sweep over detection_rate_per_sec

Reproducing the figures

git clone <this-repo>
cd <this-repo>
python -m pip install pandas plotly scipy numpy
python analyze_wvc.py --input data --output docs

Reproducing the raw data

The simulator wvc_simulator.py is fully self-contained and uses only standard scientific Python. Each dataset on this page can be regenerated:

python wvc_simulator.py --trials 20 --hours 4 --plot                # headline
python wvc_simulator.py --sweep spacing --trials 15 --hours 2       # spacing sweep
python wvc_simulator.py --sweep size --trials 15 --hours 2          # size sweep
python wvc_simulator.py --sweep detection_rate --trials 15 --hours 2 # sensitivity sweep
